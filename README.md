# API CONTATOS:

## Executar o projeto

### Instalar as dependências do projeto:

`npm i`

### Iniciar a api:

`npm run dev`

### A aplicação estará acessível na porta 3000:

`http://localhost:3000`

### Documentação do Swagger:

`http://localhost:3000/api/v1/api-docs`
